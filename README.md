# Exploring Aliexpress Data from RapidAPI
## The project is about to working with Rapid Api and consume data from Aliexpress plataform

### To Use the project after you clone it, you need to rename the .env_sample to .env and put your api key from RapidApi 
### To remember, you need to be subscribed to Magic Aliexpress
### When we have cloned the repository, We can execute the code below 
```py
pip install -r requirements.txt
```
### This is going to install all the dependencies
## To Run 
### You only need to open ali_api.ipynb in a jupyter notebook
